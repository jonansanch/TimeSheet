# Story 2.5: Eliminar Registro Propio

Status: done

## Story

As a empleado,
I want eliminar un registro propio cuando corresponda,
So that pueda corregir capturas equivocadas dentro de mis permisos.

## Acceptance Criteria

1. Dado un empleado viendo su historial, cuando selecciona eliminar un registro propio, entonces el sistema muestra una confirmación específica del registro (fecha visible), y la acción destructiva aparece claramente diferenciada (botón rojo).
2. Dado el empleado confirma la eliminación, cuando el registro pertenece al usuario autenticado, entonces el registro queda eliminado (hard delete) y deja de aparecer en el historial personal.
3. Dado un empleado intenta eliminar un registro que no le pertenece, cuando la solicitud llega al API, entonces el sistema rechaza la acción con 403 Forbidden en formato Problem Details.

### Command Handler — Patrón de Implementación

```csharp
using KPG.Timesheet.Application.Common.Exceptions;
using KPG.Timesheet.Application.Common.Interfaces;
using Microsoft.EntityFrameworkCore;

namespace KPG.Timesheet.Application.Features.RegistroHoras.Commands.DeleteRegistroHoras;

public class DeleteRegistroHorasCommandHandler : IRequestHandler<DeleteRegistroHorasCommand>
{
    private readonly IApplicationDbContext _context;
    private readonly IUser _user;

    public DeleteRegistroHorasCommandHandler(IApplicationDbContext context, IUser user)
    {
        _context = context;
        _user = user;
    }

    public async Task Handle(DeleteRegistroHorasCommand request, CancellationToken cancellationToken)
    {
        var registro = await _context.RegistrosHoras
            .FirstOrDefaultAsync(r => r.Id == request.RegistroId, cancellationToken);

        if (registro is null)
            throw new NotFoundException($"RegistroHoras con id '{request.RegistroId}' no fue encontrado.");

        if (registro.UserId != _user.Id)
            throw new ForbiddenAccessException();

        _context.RegistrosHoras.Remove(registro);
        await _context.SaveChangesAsync(cancellationToken);
    }
}
```

**Nota sobre `IRequest` sin tipo de retorno:** En MediatR, `IRequest` (sin genérico) retorna `Unit`. El handler debe ser `IRequestHandler<DeleteRegistroHorasCommand>` (no genérico), que equivale a `IRequestHandler<DeleteRegistroHorasCommand, Unit>`.

public async Task<bool> DeleteAsync(int id, CancellationToken cancellationToken = default)
{
    if (string.IsNullOrWhiteSpace(_authState.AccessToken))
        return false;

    using var httpRequest = new HttpRequestMessage(HttpMethod.Delete, $"api/registros-horas/{id}");
    httpRequest.Headers.Authorization = new AuthenticationHeaderValue("Bearer", _authState.AccessToken);

    var response = await _http.SendAsync(httpRequest, cancellationToken);
    return response.StatusCode == System.Net.HttpStatusCode.NoContent;
}
```

private async Task ConfirmarEliminarAsync(HistorialRegistroResponse item)
{
    _pendiente = item;
    StateHasChanged();
    var confirmed = await _msgBox!.ShowAsync();
    if (confirmed == true)
        await EliminarAsync(item);
    _pendiente = null;
}

private async Task EliminarAsync(HistorialRegistroResponse item)
{
    _isDeleting = true;
    _errorMessage = null;
    _successMessage = null;

    try
    {
        var success = await RegistroHorasRepository.DeleteAsync(item.Id);
        if (success)
        {
            _registros.Remove(item);
            _successMessage = $"Registro del {item.FechaRegistro:dd/MM/yyyy} eliminado.";
        }
        else
        {
            _errorMessage = "No fue posible eliminar el registro. Intente nuevamente.";
        }
    }
    catch
    {
        _errorMessage = "Ocurrió un error al eliminar el registro.";
    }
    finally
    {
        _isDeleting = false;
    }
}
```
