# Story 5.6: Reportes para Gerente y Admin

Status: review

## Story

As a gerente or admin,
I want generar reportes según mi alcance de rol,
So that pueda analizar horas desde una perspectiva gerencial o global.

## Acceptance Criteria

1. **Given** un Gerente autenticado, **when** genera reportes, **then** tiene las mismas capacidades del Supervisor **and** su alcance corresponde a los datos permitidos para Gerente.

2. **Given** un Admin autenticado, **when** genera reportes, **then** puede usar cualquier combinación de filtros globales **and** no queda limitado al equipo de un supervisor.

3. **Given** un usuario sin permisos, **when** intenta acceder al endpoint de reportes, **then** el API rechaza la solicitud.
