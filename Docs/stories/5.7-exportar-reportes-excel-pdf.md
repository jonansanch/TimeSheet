# Story 5.7: Exportar Reportes a Excel y PDF

Status: review

## Story

As a supervisor,
I want exportar reportes en Excel y PDF,
So that pueda entregar información digital lista para facturación.

## Acceptance Criteria

1. **Given** un reporte generado con filtros válidos, **when** el usuario selecciona Exportar Excel, **then** el sistema genera un archivo `.xlsx` con los datos filtrados **and** usa MiniExcel.

2. **Given** un reporte generado con filtros válidos, **when** el usuario selecciona Exportar PDF, **then** el sistema genera un archivo PDF legible **and** usa QuestPDF.

3. **Given** una exportación en curso, **when** el archivo se está generando, **then** la UI muestra estado de procesamiento **and** maneja errores sin perder los filtros seleccionados.

## Tasks / Subtasks

### Backend

- [ ] Crear `Application/Features/Reportes/Queries/ExportarReporteHoras/ExportarReporteHorasQuery.cs`
  - Mismo set de filtros que `GetReporteHorasQuery`
  - Formato: `ExportFormato.Excel` | `ExportFormato.Pdf`
  - Retorna `(byte[] Contenido, string ContentType, string FileName)`
- [ ] Crear handler en Infrastructure (usa Dapper para obtener datos, MiniExcel para Excel, QuestPDF para PDF):
  - Excel: `MiniExcel.SaveAsByteArray(items)` o stream equivalente
  - PDF: QuestPDF Document con tabla de datos, header con filtros aplicados y fecha de generación
- [ ] Agregar endpoints:
  - `GET /api/reportes/horas/excel?...` → `File(bytes, "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet", "reporte.xlsx")`
  - `GET /api/reportes/horas/pdf?...` → `File(bytes, "application/pdf", "reporte.pdf")`

### Frontend

- [ ] Agregar botones "Exportar Excel" y "Exportar PDF" en el ToolBarContent del MudDataGrid de reportes
- [ ] Al hacer clic: llamar al endpoint de export con los filtros actuales, descargar el archivo usando `IJSRuntime` (download programático via `URL.createObjectURL`)
- [ ] Deshabilitar botones durante exportación, mostrar spinner
- [ ] Manejar error de exportación con MudAlert

### Verificación

## Dev Notes

### MiniExcel export

```csharp
using var ms = new MemoryStream();
await MiniExcel.SaveAsAsync(ms, items);
return ms.ToArray();
```

### QuestPDF setup

QuestPDF requiere licencia en producción. Para V1 usar licencia Community (gratis para proyectos <1M USD ingresos):
```csharp
// En Program.cs o DI:
QuestPDF.Settings.License = LicenseType.Community;
```

Estructura básica PDF:
```csharp
Document.Create(container => container
    .Page(page => {
        page.Header().Text($"Reporte de Horas — {desde:dd/MM/yyyy} a {hasta:dd/MM/yyyy}");
        page.Content().Table(table => { /* columnas */ });
        page.Footer().Text($"Generado el {DateTime.Now:dd/MM/yyyy HH:mm}");
    }));
```

### Descarga en Blazor WASM

```csharp
// Usar HttpClient para obtener el stream y JSInterop para trigger download
var bytes = await Http.GetByteArrayAsync(url);
await JS.InvokeVoidAsync("downloadFile", fileName, contentType, Convert.ToBase64String(bytes));
```
Agregar función `downloadFile` en `wwwroot/index.html`.
